# 💰 Expense Tracker Web App

A simple web-based application to record and visualize daily expenses.

---

## ✨ Features
- Add expense transactions (item name, amount, category)  
- Display transactions in a clean card layout  
- Delete transactions easily  
- Show total expense balance  
- Visualize spending using a pie chart  

---

## 🛠️ Technologies Used
- HTML  
- CSS  
- JavaScript  
- Chart.js  

---

## 🚀 How to Run
1. Download or clone this project  
2. Open the folder in VS Code  
3. Run `index.html` in your browser  
   or use **Live Server**  

---

## 🎯 Purpose
This project is created for learning purposes and to demonstrate a simple expense tracking system with data visualization.
